¿Qué es npm?

(Node Package Manager)

Es el administrador de paquetes oficial para el ecosistema de Node.js. Node.js es un entorno de ejecución de JavaScript en el lado del servidor que permite a los desarrolladores construir aplicaciones web y otras aplicaciones de red escalables y de alto rendimiento utilizando JavaScript.

pm se utiliza para instalar, administrar y compartir paquetes de código reutilizable de JavaScript. Estos paquetes pueden contener bibliotecas, frameworks, módulos, herramientas y cualquier otro tipo de código que se pueda utilizar en proyectos de Node.js. Los paquetes pueden ser creados por desarrolladores individuales, equipos de desarrollo o incluso empresas, y se alojan en el registro público de npm para que otros desarrolladores puedan acceder a ellos y utilizarlos en sus propios proyectos.

¿Cómo inicializar un paquete?

  • Especificando preferencias
    • npm init
  • Con los valores por defecto
    • npm init -y

¿Cómo instalar paquetes?

  • Paquetes de producción - (se requieren para que funcione la aplicación)
    • npm i nombrePaquete
    npm install nombrePaquete
  • Paquetes de desarrollo:
    • npm i nombrePaquete -D
